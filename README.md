# Frontend-Misterio

## Front installation to node.js 14.x

### Check your node.js version

```
node --version
```

If you dont have, follow this [tutorial](https://computingforgeeks.com/install-node-js-14-on-ubuntu-debian-linux/)

### First time

```
$ cd Misterio/
$ npm install
```

### Run
```
$ npm start
```
